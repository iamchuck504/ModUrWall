# Animations Folder

Coloca estos 9 archivos GIF aquí:

1. **NeuralNetwork.gif** - Red Neuronal
2. **Blockchain.gif** - Blockchain  
3. **QuantumComputing.gif** - Computación Cuántica
4. **Cybersecurity.gif** - Ciberseguridad
5. **DigitalMatrix.gif** - Matriz Digital
6. **DataVisualization.gif** - Visualización de Datos
7. **digitalSamurai.gif** - Samurái Digital
8. **CloudComputing.gif** - Computación en la Nube
9. **Infrastructure.gif** - Infraestructura

## Instrucciones Rápidas

Si tienes la carpeta "Animations/" con estos archivos:

```bash
# Copia solo estos 9 GIFs necesarios
cp Animations/NeuralNetwork.gif assets/animations/
cp Animations/Blockchain.gif assets/animations/
cp Animations/QuantumComputing.gif assets/animations/
cp Animations/Cybersecurity.gif assets/animations/
cp Animations/DigitalMatrix.gif assets/animations/
cp Animations/DataVisualization.gif assets/animations/
cp Animations/digitalSamurai.gif assets/animations/
cp Animations/CloudComputing.gif assets/animations/
cp Animations/Infrastructure.gif assets/animations/
```

Luego ejecuta:
```bash
flutter pub get
flutter run
```

## Nota

Si los archivos GIF no existen, la app mostrará gradientes de colores temáticos como placeholder que se ven increíbles.

## Formatos soportados

- .gif (animados) ← **Recomendado**
- .png
- .jpg
- .webp

Se recomienda usar GIFs optimizados para mejor rendimiento en dispositivos móviles.
